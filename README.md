# InnerG Terminal — OpenClaw Dashboard

A Matrix-style terminal interface for OpenClaw, designed for the InnerG brand.

## Features

- **Matrix Rain Background** — Animated falling code effect
- **Terminal Aesthetic** — Black & green, scanlines, glowing text
- **InnerG Branding** — Logo, tagline, brand colors
- **Responsive Design** — Works on desktop and mobile
- **Session Management** — Multiple conversations
- **Export Chat** — Download conversation history
- **Toggle Effects** — Turn Matrix rain on/off

## Live Demo

Deploy on GitHub Pages or Vercel.

## Quick Deploy

1. Push to GitHub
2. Enable GitHub Pages (Settings → Pages → main branch)
3. Access at `https://[username].github.io/[repo-name]/`
